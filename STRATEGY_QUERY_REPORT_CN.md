## 2. 策略核心概述

这是一个专为**A股日线交易**设计的量化策略，综合使用以下技术面工具：

- **Squeeze Momentum Indicator (SQZMOM)** - 识别压缩点和动量反转
- **布林带 (Bollinger Bands)** - 识别盘整区间上下边界
- **Keltner Channel (KC)** - 识别趋势区间上下边界
- **指数移动平均线 (EMA)** - 多级趋势确认 (20/60/200周期)
- **平均真实波动幅度 (ATR)** - 动态止损和仓位管理
- **成交量分析** - 入场确认和虚假突破过滤

## 4. 交易逻辑详解

### 4.1 入场条件 (AND逻辑)

策略同时满足以下条件才会产生入场信号：

#### a) 趋势确认
```
EMA20 > EMA60 > EMA200 AND close > EMA20
```
- 确保处于明确的上升趋势
- 价格在所有关键EMA之上

#### b) SQZMOM 压缩
```
有效的压缩状态 (3-10根K线内的压缩)
SQZMOM值 > 0 (正动量)
```
- 识别从盘整到突破的时刻
- 压缩柱数在合理范围内

#### c) 成交量确认
```
volume > 20日均成交量 × 1.2
```
- 突破需要成交量支撑
- 避免无量突破

#### d) 综合评分
```
评分 >= 65分
```

#### e) 风控过滤
- ✅ 非涨停板走势
- ✅ 非ST股票

- ✅ 未超额反弹 (涨幅<30%)

- ✅ 无虚假突破 (上影线占比<60%)

### 4.3 出场逻辑 (4级出场)

出场按优先级顺序触发，第一个满足的条件生效：

#### 第一阶段: 部分获利 (TP1)
```
触发价格: TP1 = 入场价 + 止损距离
操作: 平仓50%仓位 (保留50%)
特点: 只触发一次
```

#### 第二阶段: 追踪止损 (ATR Trail)
```
止损线: 最高价 - ATR × 2.5
触发: close ≤ 追踪止损线
操作: 平仓全部仓位
```

#### 第三阶段: EMA20止损
```
触发: close < EMA20
操作: 平仓全部仓位
```

#### 第四阶段: 二级目标 (TP2)
```
触发价格: TP2 = 入场价 + 止损距离 × 2
操作: 平仓全部仓位
```

## 5. 仓位和风险管理

### 5.1 仓位计算公式

```
风险金额 = 账户权益 × 风险百分比 / 100
止损距离 = ATR × ATR倍数 (2.5)
仓位数量 = max(1, floor(风险金额 / 止损距离))
```

### 5.2 例子 (账户10万, 风险1%)

假设当前指标值：
- 账户权益: ¥100,000
- ATR(14周期): ¥50
- 风险百分比: 1%

计算过程：
1. 风险金额 = 100,000 × 1% = ¥1,000
2. 止损距离 = 50 × 2.5 = ¥125
3. 仓位 = floor(1,000 / 125) = **8手**

### 5.3 滑点处理

入场价 = close + (ATR × 0.1)
- 加入向上滑点，更真实地模拟实盘
- 滑点 = ATR的10%

### 8.2 关键函数

#### f_sqz(src) 函数
```pine
f_sqz(src)=>
    basis = ta.sma(src, bbLen)           // 布林带基准线
    dev   = bbMult * ta.stdev(src, bbLen) // 布林带偏离

    upperBB = basis + dev                // 布林带上轨
    lowerBB = basis - dev                // 布林带下轨

    ma = ta.sma(src, kcLen)              // KC中线
    kcATR = ta.atr(kcLen)

    upperKC = ma + kcATR * kcMult        // KC上轨
    lowerKC = ma - kcATR * kcMult        // KC下轨

    // 压缩信号: BB在KC内部
    sqzOff = lowerBB < lowerKC and upperBB > upperKC

    mid = (ta.highest(high,kcLen)+ta.lowest(low,kcLen))/2
    val = ta.linreg(src-mid,kcLen,0)    // 动量值

    [sqzOff,val]                        // 返回压缩信号和动量
```
