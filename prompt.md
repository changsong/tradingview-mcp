## 研报及新闻分析
1. 利用 webNews.js/usNew.js 获取这 11 只推荐股票的新闻，包含研报，快讯，新闻等在内的信息，进行 市场情绪倾向（看涨/看跌/中性比例），对股价的潜在影响预测（利好/利空因素），关键风险和机会点
   请将新闻转化为“可交易信号”，最好最近3天最重要新闻。
【任务】
    1. 抓取并筛选高影响力新闻（去噪）
    2. 对每条新闻打标签：
      - 类型：政策 / 财报 / 并购 / 行业 / 黑天鹅 / 传闻
      - 情绪：+1 / 0 / -1
      - 影响权重：1~5
    3. 构建情绪指数：
       - Sentiment Score = Σ(情绪 × 权重)
    4. 识别关键模式（重点）
       - 情绪持续增强（trend）
       - 情绪反转（reversal）
       - 情绪背离（price vs news）
    4.输出交易信号：
       - Long / Short / No Trade
       - 触发原因（必须可解释）
        - 是否适合：追涨/低吸/反转
    【重点】
    必须识别：
     - 是否已经提前炒作
     - 是否存在情绪过热
     - 是否是假利好
【输出要求】
同时输入JSON格式输出，字段清晰，可直接用于程序读取
