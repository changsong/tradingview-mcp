### 1. 趋势结构分析（Trend Structure）

#### 指标组合
| 指标 | 看涨信号 | 看跌信号 | 中性 |
|------|---------|---------|------|
| **价格 vs SMA20** | 价格 > SMA20 | 价格 < SMA20 | 价格 ≈ SMA20 |
| **SMA 排列** | 20>50>200 | 20<50<200 | 混乱排列 |
| **EMA 12vs26** | EMA12>EMA26 | EMA12<EMA26 | 交叉状态 |

#### 应用规则
```
STRONG UPTREND (强上升):
  条件：价格>SMA20 AND SMA20>SMA50 AND EMA12>EMA26
  强度：+3
  交易：可追涨（但需配合动能）

UPTREND (上升):
  条件：价格>SMA20 AND SMA20>SMA50
  强度：+2
  交易：低吸机会 > 追涨

CONSOLIDATION (盘整):
  条件：SMA 排列混乱
  强度：0
  交易：观望，等待方向确认
```

### 2. 动能系统（Momentum System）

#### RSI(14) 应用
```
RSI > 70：超买
  • 风险：回调概率 60%+
  • 操作：停止追入，设置减仓计划

RSI 60-70：强势（但非超买）
  • 风险：可控
  • 操作：继续持有或轻仓加仓

RSI 50-60：健康水平
  • 风险：低
  • 操作：正常交易

RSI 40-50：温和
  • 风险：低
  • 操作：可以逢低建仓

RSI 30-40：弱势（但非超卖）
  • 风险：可控
  • 操作：等待反弹后再做

RSI < 30：超卖
  • 机会：反弹概率 60%+
  • 操作：分批低吸
```

#### MACD 应用
```
MACD > Signal Line AND 直方图 > 0：看涨
  • 强度：由直方图大小决定
  • 操作：可追入

MACD < Signal Line OR 直方图 < 0：看跌
  • 强度：由直方图大小决定
  • 操作：减仓或回避

黄金叉 (Golden Cross)：
  • MACD 从下穿破 Signal Line
  • 信号强度：中等（需要成交量确认）
  • 操作：可视为买点（结合其他指标）

死叉 (Death Cross)：
  • MACD 从上穿破 Signal Line
  • 信号强度：中等
  • 操作：减仓或止损信号
```

### 3. 波动性分析（Volatility）

#### ATR(14) 应用
```
ATR 高 (波动率 > 平均的 20%)：
  • 含义：高风险、高机会
  • 操作：止损设置要宽松（5-8%），仓位要小

ATR 低 (波动率 < 平均的 5%)：
  • 含义：波动压缩，可能酝酿方向
  • 操作：准备突破入场，止损可紧（3-5%）

ATR 中等 (平均水平)：
  • 含义：正常环境
  • 操作：标准止损设置（-5% 左右）
```

#### 布林带（Bollinger Bands）应用
```
价格 > 上轨：过热，回调风险
  • 操作：不追入，等待回调

价格 < 下轨：过冷，反弹机会
  • 操作：可低吸

布林带宽 < 15%：压缩（Squeeze）
  • 含义：酝酿突破
  • 操作：准备突破交易（需要成交量配合）

布林带宽 > 50%：扩张
  • 含义：波动加大，可能突破在进行
  • 操作：追入但要控制风险
```

### 4. 成交量行为（Volume）

#### 核心原则
```
成交量确认价格：
  ✅ 价格上升 + 成交量扩张 → 看涨有效
  ✅ 价格下跌 + 成交量扩张 → 看跌有效
  ❌ 价格上升 + 成交量萎缩 → 虚弱上升（风险）
  ❌ 价格下跌 + 成交量萎缩 → 虚弱下跌（反弹风险）

成交量倍数：
  1.3x 平均：温和扩张
  1.5x 平均：明显扩张
  2x+ 平均：强烈成交（突破信号，或出货信号）
```

#### 应用规则
```
突破确认：
  • 价格突破关键位置 + 成交量 > 1.5x 平均
  → 突破有效，可追入

虚假突破警告：
  • 价格突破 + 成交量萎缩 → 假突破可能
  → 等待回落后再入

底部形态确认：
  • 价格在关键支撑反弹 + 成交量爆增
  → 底部确认，可建仓
```
