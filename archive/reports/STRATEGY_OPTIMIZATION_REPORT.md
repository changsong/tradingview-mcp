## 📊 优化内容

### 新增三个过滤条件

1. **前10日大涨过滤**
   - 检查前10个交易日内任意单日涨幅
   - 默认阈值: 20%（超过则过滤）
   - 目的: 避免在大涨后追高，减少回调风险

2. **26周低位过滤**
   - 计算当前价格在26周（130个交易日）区间的位置
   - 默认阈值: 70%（必须低于此位置）
   - 目的: 确保在相对低位入场，提高上涨空间

3. **5日波动过滤**
   - 计算最近5个交易日的价格波动幅度
   - 默认阈值: 20%（波动不超过此值）
   - 目的: 避免波动过大的股票，降低风险

### 优化开关

- 新增 `enableOptimization` 参数，可以一键启用/禁用优化过滤
- 参数可调整，适应不同市场环境

## 💡 优化建议

### 1. 参数调整建议

基于BHRB的成功案例，当前参数设置较为合理：

```pine

lowPositionPct = 70.0     // 26周低位阈值%
max5dVolatility = 20.0    // 5日最大波动%
```

### 2. 进一步优化方向

#### 方向A: 放宽条件，增加交易机会

```pine
maxSurge10d = 25.0        // 放宽至25%
lowPositionPct = 80.0     // 放宽至80%
max5dVolatility = 25.0    // 放宽至25%
```

**优点**: 可能捕获更多交易信号
**缺点**: 可能降低胜率

#### 方向B: 收紧条件，提高胜率

```pine
maxSurge10d = 15.0        // 收紧至15%
lowPositionPct = 60.0     // 收紧至60%
max5dVolatility = 15.0    // 收紧至15%
```

**优点**: 可能提高胜率和盈利因子
**缺点**: 交易机会减少

#### 方向C: 调整止损止盈

当前设置:

建议测试:
